Super Scroll List For NGUI

Table support!
It can help to build a list of reused elements with different 
size in high-performance.
-- version 2.0


This plugin is an extension tool for NGUI scroll view, 
it can help you to spawn a list of elements in a scroll 
view and manager them, it has a stable high-performance 
nomatter how much elements in the list.

Please run example scenes before you use it, they are 
all in the "SuperScrollList/Examples" folder.

This plugin requires NGUI 3.7.0 or newer version, please 
make sure your NGUI is suitable.
-- version 1.0
